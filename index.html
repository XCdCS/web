Basic Programming

After completing web development, we're going to move to pseudocode and flowcharts. Pseudocodes are english-like representations of algorithms, while flowcharts are visual representations using a diagram. Both are used to make code easier to understand. Note: pseudocode is not written in any specific programming language.
